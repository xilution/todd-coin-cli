# todd-coin-cli

Todd Coin is a Cryptocurrency for Good.

This is the Todd Coin CLI.

## First Things First

`npm install`

## To Build

`npm run build`

## To Watch

Hot rebuild when `src` changes

`npm run watch`

## To Install todd-coin CLI Locally

`npm install -g .`

## To See todd-coin CLI Commands and Options

`npm run build && todd-coin`

## To Release

`npm run release`
